# Planning Decisions

Decisions made during planning sessions that inform future triage and scope.
The planning agent reads this file before classifying TRIAGE NEEDED items,
and proposes updates to it after each approved plan. Do not edit this file
manually — let the agent maintain it and review its proposed changes.
